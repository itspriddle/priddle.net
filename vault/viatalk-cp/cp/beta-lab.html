VT Beta Lab

What Is The Beta Lab?

This page allows you to take a look at the new features that we are developing to improve your service. It will list the features that we are currently beta testing and will show (for a little while) the features that moved from beta testing to full launch. You can also submit suggestions for new features you would like to see for your future phone service.

Beta Testing

This section shows you what features have been released for limited use. These features are currently being tested and will be rolled out once bugs are identified and solved.

Features marked by are currently in beta. If you believe you have identified a bug in a beta feature, please submit a bug report to our developers.

  • Just Added Incoming Routing - Routes can now be enabled/disabled individually

Full Release

Once the feature is released for all ViaTalk Plans, it will be taken from the Beta Testing Section and placed here as well as in the Control Panel itself. These features will remain in this section for a short while as a reminder of new features that have just been developed and released for full use.

  • Voicemail Notifications now available via AOL Instant Messenger.
  • Customizable Notification Formats let you change the text formatting on your voicemail alerts.
  • Blacklist by Caller ID allow you to block incoming calls by name using full or partial text matches.
  • Outgoing Call Routing lets you control what happens when certain numbers are dialed. Examples: Map 311 to a local number or block the kids from calling certain numbers.
  • Pattern Match Filters/DND Whitelist - Whitelist numbers that are blocked via Pattern Match Filters, or Do Not Disturb.
  • Incoming Call Routing now gives you the options to forward calls to a generic disconnect message or have specific numbers simultaneously ring to the number of your choosing.
  • View Call Logs for your Virtual Numbers
